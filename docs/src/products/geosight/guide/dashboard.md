# Navigating the Dashboard

## Home Page Layout

When you open GeoSight, you’ll land on the <span class="ui-page-label">Home</span> page, which acts as the central hub for accessing dashboards, projects, and platform features. The layout is simple and focused, allowing you to quickly find and open projects or navigate to administrative tools.

<br>

It contains the following key sections:

<table class="my-table-style">
  <thead>
    <tr>
      <th>Section</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Banner</td>
      <td>Displays the main header and navigation controls, including access to the home page.</td>
    </tr>
    <tr>
      <td>Project Gallery</td>
      <td>Shows available dashboards, including your own, shared, and public projects.</td>
    </tr>
    <tr>
      <td>Admin / User</td>
      <td>Provides access to login/logout, account settings, and administrative tools.</td>
    </tr>
  </tbody>
</table>

### Project Gallery

The <span class="ui-page-label">Project Gallery</span> displays all dashboards available to you in one place. A project is a collection of indicators, context layers, widgets, and filters, allowing you to analyse and understand conditions within a specific area.

<br>

<div class="alert alert-note">
  <div class="alert-icon">📝</div>
  <div class="alert-text">
    The projects you see will depend on your access level and whether you are logged in.
  </div>
</div>

<br>

Each project is displayed as a <span class="ui-generic-label">Project Card</span> with the following elements:

<table class="my-table-style">
  <thead>
    <tr>
      <th>Element</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Thumbnail</td>
      <td>A visual preview representing the theme or content of the project.</td>
    </tr>
    <tr>
      <td>Title</td>
      <td>The name of the project.</td>
    </tr>
    <tr>
      <td>Description</td>
      <td>A short project summary. If too long, it can be scrolled within the card.</td>
    </tr>
  </tbody>
</table>

<br>

## Administrator Panel

The <span class="ui-page-label">Administrator Panel</span> is where you create and manage content within GeoSight. It allows you to manage data, build dashboards, configure layers, and control user access.

Users have full control over the data, projects, and layers they own. Editable items are highlighted in blue text, while non-editable items appear in black.
